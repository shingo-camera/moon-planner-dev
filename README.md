# MOON × LANDMARK PLANNER

PLの塔・岐阜城・和歌山城・高津子山展望台と月（または太陽）の撮影計画を支援するWebツールです。  
インストール不要の単一HTMLファイルで動作します。

---

## 機能一覧

### 01 対象選択
- **PLの塔**（大阪府富田林市）/ **通天閣**（大阪府大阪市）/ **和歌山城**（和歌山県和歌山市）/ **高津子山展望台**（和歌山県和歌山市）/ **岐阜城**（岐阜県岐阜市）を切り替え
- **☀️ 太陽モード**：月の代わりに太陽（日の出・日没）との構図を計画。全対象で有効
- 太陽モード中は満月チャンス検索・満ち欠けカレンダーは非表示

### 02 日時設定
- 起動時に現在の日時（JST）を自動セット
- 日付フィールドに曜日表示（日=赤、土=青）
- **±ボタン**で時刻を微調整（±1 / ±5 / ±10 / ±60分）
- **NOW**ボタンで現在時刻を即セット
- 日の出・日の入り・月の出・月の入りの時刻をクリックするとその時刻をセット

### 03 焦点距離・三脚高さ
- スライダーまたはプリセットボタンで焦点距離を設定（9〜1600mm、12段階）
- 横位置・縦位置の切替対応（35mmフルサイズ基準）
- **三脚高さ**：±1 / ±5cmボタンで調整（範囲：10〜300cm、デフォルト150cm）
- **Rst**ボタンでデフォルト（150cm）にリセット
- 三脚高さはプレビューの仰角計算に反映（超望遠時に特に有効）

### 04 撮影地点の指定
3種類の方法で撮影地点を指定できます：

| 方法 | 操作 |
|---|---|
| 地図タップ/クリック | 地図上の任意の地点をタップ/クリック |
| 📍 現在地 | GPSで現在地を自動取得（ブラウザの位置情報許可が必要） |
| 📋 座標ペースト | Googleマップで地点を長押し→緯度経度をタップしてコピー→📋ボタンで貼り付け |

**📋ボタンの入力形式**：
- `34.2283844, 135.1500548` 形式（Googleマップの緯度経度コピー）
- 度分秒形式（`34°29'58.3"N 135°32'37.9"E`）
- Google マップURL各種（短縮URL・通常URL・Place URL・データURL）

指定後に自動で以下を取得・計算：
- 撮影地点の標高（国土地理院API）
- 対象までの水平距離・3D距離・方位角・仰角

**撮影地点情報の確認（ドロワー04）**：
- 距離・標高・方位角・仰角をリアルタイム表示
- **標高±ボタン**（±1 / ±3 / ±5m）で標高を補正可能。屋上・堤防など地面より高い場所からの撮影に使用
- **Rst**ボタンで補正をリセット

**地図操作**：
- 撮影地点マーカー（青丸）はドラッグで移動可能
- 右上の🛰ボタンで地図／衛星画像（Esri World Imagery）を切替
- 右下の🗺ボタンでGoogleマップを撮影地点座標で開く
- 右下の👁ボタンでGoogleストリートビューを撮影地点から対象方向を向いた状態で開く

### 05 太陽・月情報
- 日の出 / 日の入り / 月の出 / 月の入り時刻を表示
- 月齢・月相名・月相アイコンを表示
- 各時刻をクリックすると設定時刻に反映

### ☁️ 天気情報（Open-Meteo API）
撮影地点を設定済みで、選択日時が**現在から16日以内**の場合に天気情報を自動取得・表示します。

**表示場所**：
- **ドロワーヘッダー**：日時の下に ☔雨% 👁透% / ☁️低% 中% 高% を表示
- **スマホ縦向きダイアグラム**：コンパス円の右下に4行で表示
- **満ち欠けカレンダー**：月の出・入り時刻の下に「雨%透%」を表示
- **チャンス一覧（90日・満月）**：16日以内の結果に ☔%👁%☁️低中高 を表示

**天気項目**：
- ☔ **雨%**：降水確率（%）
- 👁 **透%**：透明度（視程÷視線距離×100）。撮影地点から対象方向の見通しの良さを示す
- ☁️ **低/中/高**：低層（〜2km）・中層（2〜6km）・高層（6km〜）の雲量（%）

**取得仕様**：
- データソース：[Open-Meteo](https://open-meteo.com/)（無料・無認証）
- 16日先まで・1時間ごとのデータ
- キャッシュ：座標キーごとに1時間保持（同一地点は再取得しない）
- 取得トリガー：撮影地点変更・対象切替・太陽モード切替・ドロワーを開いたとき・時刻変更時

### 06 チャンス一覧
「📅 選択日から90日を検索」ボタンで、現在設定中の撮影地点から対象に天体（月または太陽）が**物理的に重なる日時**を自動検索します。

**判定ロジック（月モード）**：
- 1分刻みで全時刻をスキャン（90日分）
- 月の方位角が対象方位角と一致する範囲内
- 月の仰角が対象の底辺〜頂点の仰角範囲内
- 輝面比5%未満（ほぼ新月）の日は除外
- 建造物頂点の仰角に最も近い瞬間を記録

**判定ロジック（太陽モード）**：
- 日の出後・日の入り前の時刻を対象に検索

**結果表示**：
- 🎯🌕🎯：横移動5m以内（超レアチャンス）
- ★★★：10m以内 / ★★☆：50m以内 / ★☆☆：100m以内 / ☆☆☆：200m以内 / ―：圏外
- 月相アイコン・月齢・横移動距離（移Xm）・建物内高さ位置（高X%）を表示
- 撮影地点設定済み・選択日から16日以内の結果には天気情報（☔雨% 👁透% ☁️低/中/高）を表示
- クリックで日時を一括セット → プレビューで目視確認

> チャンス一覧は絞り込み用の目安です。最終確認はプレビューで行ってください。

### 07 満月チャンス（月モードのみ）
「🌕 満月チャンス（1年）を検索」ボタンで、月齢13〜16.5日（満月前後±1.5日）に絞って1年間のチャンスを検索します。16日以内の結果には天気情報も表示されます。

### プレビュー
- 指定焦点距離・時刻での空のプレビューをリアルタイム表示
- 対象シルエットと月または太陽を重ねて描画
- 月は月齢に応じた位相を描画、月相アイコンも表示
- 太陽は仰角に応じて色が変化（低いほど赤みが強くなる）
- **軌道トレイル**：現在時刻の前後の天体位置を時刻付きの点で表示
- **ピンチズーム**：2本指ピンチで拡大・縮小（最大5倍）。拡大時は1本指ドラッグでパン可能。ダブルタップでリセット
- **視点オフセット（⊹ボタン）**：カメラの向きを上下左右に微調整して構図を変更。建造物を画面中心からずらした構図のシミュレーションに使用
- 地球曲率補正実装済み（R=6371km）

### 方位・仰角ダイアグラム
- 撮影地点から見た対象・天体の方位を円形ダイアグラムで表示
- 方位角・仰角・月齢（月相）を数値表示
- スマートフォン縦向き時：ダイアグラム内に天気情報（☔雨% 👁透% / ☁️低/中/高）を表示

### 撮影候補エリア（地図上の扇形）
- 天体の出後30分・入り前30分の扇形を表示
- 同心円で距離目安を表示（地図ズームに連動して刻みを自動調整）

### 🌙 満ち欠けカレンダー（月モードのみ）
- 月ごとの月齢・月相アイコン・月相名・月の出入り時刻を一覧表示
- 撮影地点設定済み・16日以内の日付には月の出入り時刻下に天気情報（雨%・透%）を表示
- 日付をクリックで日付フィールドにセット

### 💾 地点保存
- 現在の撮影地点・対象・日時・焦点距離・三脚高さ・視点オフセットをまとめて保存
- 最大100件まで保存可能（ブラウザのローカルストレージ）
- 🔗ボタンでURLに変換してクリップボードにコピー（撮影計画の共有に使用）
- 保存済みスロットをクリックで設定を丸ごと復元
- **📅 登録地点90日一括ピンポイント検索（30m以内）**：登録済みの全地点に対して90日間のチャンスを一括検索。横移動30m以内に絞った精度の高い結果を各地点ごとに表示

## スマートフォン対応

- モバイル専用レイアウト（縦画面）で最適化
- **横向き禁止オーバーレイ**：横向きにすると全画面で縦に戻すよう案内を表示
- ボトムバーから時刻・焦点距離・三脚高さ・構図切替・設定（ドロワー）を操作
- ドロワー（⚙ボタン）から全設定項目にアクセス
- **📖 操作ガイド**（`guide.html`）をドロワー最下部から参照可能

---

## 使い方の流れ

1. 対象（PLの塔 / 岐阜城 / 和歌山城 / 高津子山展望台）を選択
2. 撮影予定日を設定
3. **チャンス一覧**で重なる日時を検索 → クリックで日時をセット
4. 地図の扇形エリアを参考に撮影候補地点をタップ（またはドラッグ・座標ペースト）
5. プレビューで天体と対象の重なりを目視確認
6. ±ボタンで時刻を微調整、三脚高さ・標高オフセットも必要に応じて調整
7. 視点オフセットで構図を調整
8. 地点保存で撮影計画を記録・共有

| 通天閣座標 | 34.652530°N 135.506299°E（高さ100m） |

| 大気差補正 | 標準大気（気温10℃、気圧1013hPa）の仰角依存式で自動補正 |
| 地球曲率補正 | 実装済み（R=6371km） |

## 動作環境

- モダンブラウザ（Chrome / Firefox / Safari / Edge）
- インターネット接続が必要（地図タイル・標高API・天気API・短縮URL展開）
- 位置情報・クリップボードは初回のみ許可が必要

---

## 注意事項

- プレビューはあくまで**目安**です。実際の撮影時は現地で調整してください
- 大気差は標準大気での補正値を使用しています。実際の気象条件により多少異なる場合があります
- 天気情報はOpen-Meteo APIの予報値です。16日を超える日付は表示されません。予報精度は日数が増えるほど低下します
- 短縮URLの展開には外部プロキシ（corsproxy.io / allorigins.win）を経由します
- 保存データはブラウザのローカルストレージに保存されます。ブラウザのデータを削除すると消えるため、重要な地点はURLエクスポートでバックアップしてください
